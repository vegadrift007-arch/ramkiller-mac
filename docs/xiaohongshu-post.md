```
很多人觉得 Mac 不会中毒
事实是 macOS 上的恶意软件越来越多 🚨
Shlayer / Adload / XCSSET 这些家族都很活跃

但市面上的 Mac 杀毒软件大都：
· 收费贵 💸
· 联网上传数据 📡
· 后台跑一堆服务 🐌
· 闭源不知道在干嘛 🕳️

干脆自己写了一个开源版 ↓↓↓

🦠 已知恶意软件
内置 20+ 个 Mac 恶意软件家族特征库
```

## 💡 为什么这个角度更好（思路说明）

**1. 避开拥挤赛道**
"Mac 清理"领域已经被 Pearcleaner（13k ⭐）、Tencent 柠檬（6k ⭐）、PureMac（4k ⭐）占满。我们以"清理工具"出道很难突围。

**2. 占领空白领域**
开源 macOS "杀毒/安全扫描"工具几乎没有。Patrick Wardle 的 KnockKnock 免费但不开源。这个赛道**真空**。

**3. 戳到真实焦虑**
"我的 Mac 安全吗"是个长期被忽视的问题。引发好奇 + 焦虑 → 高点击率。

**4. 顺势带出其他功能**
开头 5 个安全卖点之后再说"还附送内存清理、缓存清理"，会让人觉得**超值** — 反向用力反而比正面拼"清理"更强。
